儿童机票购买完整指南

交通出行 全年龄段适用 更新于 2026-06-15

一、票价与座位规则

  • 婴儿票（0-2周岁）：按成人全价票的10%收取，无独立座位，需大人怀抱乘机
  • 儿童票（2-12周岁）：按成人全价票的50%收取，有独立座位，可享受餐食
  • 部分特价促销时，儿童票可购买成人折扣票，需提前核对退改规则

二、所需证件

  • 国内航班：户口本、出生证明、身份证三选一即可
  • 国际航班：需办理儿童护照，部分国家需出生证明公证
  • 建议优先使用身份证，安检通关速度更快

三、婴儿车与安全座椅

  • 折叠式婴儿车可免费托运，部分航司允许推至登机口
  • 儿童安全座椅需占用座位，需购买儿童票后使用
  • 伞车类轻便婴儿车多数航司可带上客舱，需提前确认尺寸
💡 省钱小贴士

2岁生日当天按婴儿票计算，超过当天即需购买儿童票；可选择生日前出发的航班，节省一半机票费用。
