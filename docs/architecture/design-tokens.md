# 设计规范

## 颜色与阴影

| Token | 值 | 用途 |
|-------|-----|------|
| primary | `#4A90D9` | 主色：按钮、链接、品牌色 |
| accent | `#F5A623` | 强调色：预算金额、评分 |
| success | `#7EB8A0` | 成功色：已完成、亲子友好标签 |
| danger | `#EF4444` | 危险色：警告、限制提示 |
| bg-base | `#FFFBF5` | 页面背景 |
| shadow-card | `0 2px 8px rgba(0,0,0,0.06)` | 卡片默认阴影 |
| shadow-card-hover | `0 4px 16px rgba(0,0,0,0.08)` | 卡片悬浮阴影 |
| font-sans | `Inter, system-ui, sans-serif` | 全站字体 |
