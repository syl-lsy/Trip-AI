> user: webhook是什么

> assistant: Webhook 是 HTTP 的**反向回调**。简单说：

- 常规 API：你主动请求服务器 → 拿到数据
- Webhook：服务器主动向你发 HTTP 请求（当某个事件发生时）

工作流程：
```
事件发生（如：新订单）→ 你的服务器 → POST 到一个预先注册的 URL → 接收方处理
```

**在你项目中的典型场景**：

| 场景 | 触发事件 | Webhook 发送方 | 接收方 |
|------|---------|---------------|--------|
| 支付回调 | 用户支付成功 | 支付宝/微信 | 你的 NestJS 后端 |
| 邮件通知 | 行程提醒时间到 | n8n 工作流 | SendGrid/Mailgun |
