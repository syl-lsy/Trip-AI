## 1️⃣ 三亚旅行攻略真实分享第2️⃣篇（热度最高🔥 3793赞/3036藏）
**作者：是初夏小宝贝呀**

## 结论：不支持自动 `@path` 导入

> **"虽然 opencode 不会自动解析 AGENTS.md 中的文件引用"**

OpenCode **没有** CLAUDE.md 那种 `@path` 自动展开语法。你在 AGENTS.md 里写 `@docs/xxx.md`，OpenCode **不会**自动把文件内容注入进来。

## 但有两种替代方案

### 方案一（推荐）：`opencode.json` 的 `instructions` 字段

```json
{
  "instructions": [
    "docs/development-standards.md",
    "docs/api-endpoints.md",
    "docs/architecture/tech-stack.md",
    "docs/architecture/design-tokens.md",
    "docs/architecture/project-structure.md",
    "docs/mcp-services.md"
  ]
}
```

这些文件会**自动合并**到 AGENTS.md 的上下文中一起提供给 LLM。支持 glob 匹配和远程 URL。

### 方案二（手动）：AGENTS.md 中指令 AI 按需读取

```markdown
## 外部文件读取说明
当任务需要时，使用 Read 工具读取以下文件获取详细信息：
- 技术栈: docs/architecture/tech-stack.md
- 设计规范: docs/architecture/design-tokens.md
- API 端点: docs/api/api-endpoints.md
```

这种方式 AI 只在需要时才读取，不浪费上下文。

---
