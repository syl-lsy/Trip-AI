## 童行 AI（Trip with Kids）— 项目全景

**定位**：AI 驱动的亲子旅游一站式规划平台。带娃出行输入需求，AI 自动生成适合孩子年龄的完整行程。

### 技术栈
| 层 | 技术 |
|----|------|
| 前端 | Vue 3 (Composition API) + TypeScript + Tailwind CSS + Vite |
| 后端 | NestJS + Prisma + PostgreSQL (pgvector) + Redis |
| AI | LangChain + LangGraph + DeepAgent |
| 认证 | JWT + 手机验证码 |

### 项目结构
- `client/` — Vue 3 前端（views: 登录/规划/知识库/目的地/行程）
- `server/` — NestJS 后端（auth/itinerary/knowledge/destination 模块，19 API 端点）
- `ai/` — AI Agent 层（IntentRouter → TravelPlanner/TransportAgent/AccommodationAgent/AttractionAgent/BudgetAgent/TravelModifier/KnowledgeQA）
- `docs/` — 文档（PRD、agent-workflow、OpenAPI 规范、记忆系统）

### 子智能体系统
8 个子 Agent：coordinator(编排) → planner(只读分析) → frontend-dev / backend-dev / ai-dev → tester → reviewer(pro 模型) → docs-writer，形成完整开发流水线。

### 记忆系统
纯文件 5 层架构：Hot(ACTIVE-CONTEXT.md) → Warm(MEMORY.md) → Daily(daily/) → Topic(topics/) → Cold(archives/)，无向量语义索引。
